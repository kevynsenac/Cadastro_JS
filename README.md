# 📘 Atividade: Praticando Estrutura HTML e Estilização CSS

Este projeto consiste em um formulário de cadastro simplificado, desenvolvido como um exercício prático para observar como as folhas de estilo (CSS) manipulam visualmente os elementos estruturais do HTML.

## 🎯 Objetivo
O foco principal é o aprendizado da integração entre as tecnologias web fundamentais, transformando um "esqueleto" de texto em uma interface organizada e moderna (Clean Blue).

## 🛠️ Conceitos Praticados

* **Estrutura Semântica**: Organização do conteúdo utilizando tags apropriadas como `<main>`, `<form>`, `<label>` e `<input>`.
* **Estilização com CSS**:
    * **Seletores**: Aplicação de regras em seletores globais (`*`), de classe (`.container`) e de ID (`#btn-cadastrar`).
    * **Box Model**: Controle de espaçamento interno (`padding`), margens (`margin`) e bordas para definir o layout.
    * **Visual Clean**: Uso de uma paleta de cores baseada em tons de azul e branco para garantir legibilidade.
* **Lógica de Interação**: Uso de JavaScript para capturar o evento de envio (`submit`) e exibir um feedback simples ao usuário.

## 📂 Organização do Projeto

1.  **`index.html`**: Contém a marcação dos campos de entrada e botões.
2.  **`style.css`**: Define toda a identidade visual e o posicionamento dos elementos.
3.  **`script.js`**: Gerencia o comportamento do formulário após o preenchimento.
